# FiSGO
FiSGO: Finite simple groups by order, a SageMath program to determine which simple groups divide a given order
# Usage
We recommend importing the classes and functions in FiSGO_v1.1.py using the load("FiSGO_v1.1.py") command in Sagemath Notebook or Sagemath Console. Otherwise, one may use the Sagemath Notebook file (.ipynb) directly although it is not recommended.
